# portifolio-qa
Projeto: Relatório de Bugs - E-commerce (Laboratório de QA)

Descrição:
Este projeto foi desenvolvido durante a "Missão QA - Caça aos Bugs". O objetivo foi realizar testes exploratórios e funcionais em uma plataforma de e-commerce simulada, focando na identificação de falhas de lógica, validação de campos e interface.

O que eu fiz:

Analisei os principais fluxos do sistema (Cadastro, Busca, Carrinho).

Identifiquei inconsistências de validação (ex: aceitação de caracteres especiais em campos de texto).

Documentei cada falha em uma planilha profissional com ID, Severidade e Passos para Reprodução.

Ferramentas utilizadas:

Planilha de Gestão de Defeitos (Excel/Google Sheets).

Navegador (Testes de Interface).
